# Artist Portfolio Design System

## Overview

To access the artist's official brand identity and style resources, use this skill.

**Keywords**: branding, visual identity, styling, brand colors, typography, minimalist design, gallery aesthetic, artist portfolio, visual formatting, design

## Brand Guidelines

### Colors

**Main Colors:**

- Background: `#FFFFFF` - Soft, gallery-like backdrop for all pages
- Text Primary: `##000000`
- Text Secondary: `#000000` 

<!-- **Accent Colors:**

**Usage Guidelines:**
- Maintain high contrast ratios (WCAG AA minimum: 4.5:1)
- Use color sparingly to emphasize calls-to-action
- Keep artwork colors pure by using neutral backgrounds
- Preserve the gallery-like aesthetic with minimal color interference

### Typography

**Fonts:**

- **Headings:** Lato Bold 700(with Arial fallback)
- **Body Text:** Lato Light 300(with Arial fallback)
<!-- - **Accent Font** (Optional): Serif font for artist statement

- **Note:** Fonts should be pre-installed in your environment for best results


**Typography Scale:**
- H1: 48px / 3rem (Desktop), 36px / 2.25rem (Mobile)
- H2: 36px / 2.25rem (Desktop), 28px / 1.75rem (Mobile)
- H3: 28px / 1.75rem (Desktop), 24px / 1.5rem (Mobile)
- H4: 24px / 1.5rem
- Body: 18px / 1.125rem
- Small: 14px / 0.875rem
- Line Height: 1.6 for body text, 1.2 for headings

## Design Philosophy

### Core Principles

- **Minimalist Approach**: Let the artwork take center stage; reduce visual clutter
- **Intentional White Space**: Use generous spacing to create breathing room around pieces
- **Seamless Navigation**: Intuitive user experience that doesn't distract from the art
- **Responsive Design**: Beautiful presentation across all devices and screen sizes
- **Performance First**: Fast loading times to maintain engagement
